# Lockstride Claude Code Marketplace

Official Claude Code plugin marketplace for [Lockstride](https://lockstride.ai).

## Available Plugins

### Lockstride Kickoff

AI-guided workflow for startup ideation and development - from concept brainstorming to product handoff.

**Documentation:** [lockstride/kickoff](https://github.com/lockstride/kickoff)

## Installation

### Add the Marketplace

```
/plugin marketplace add lockstride/claude-marketplace
```

### Install Lockstride Kickoff

```
/plugin install lockstride-kickoff@lockstride-marketplace
```
